# RuBypass

Приложение для управления сетевыми маршрутами. Позволяет выборочно направлять определённые IP-диапазоны через шлюз локальной сети, не затрагивая остальной трафик.

Работает в системном трее — никаких лишних окон.

---

## Установка

### macOS (Intel и Apple Silicon)

1. Скачайте `RuBypass_x.x.x_universal.dmg` из [последнего релиза](https://github.com/iamcoop1989-cell/rubypass/releases/latest)
2. Откройте DMG и перетащите RuBypass в папку «Программы»
3. При первом запуске: ПКМ → «Открыть» → «Открыть» (обход Gatekeeper)
4. Значок появится в строке меню

### Windows (x64)

1. Скачайте `RuBypass_x.x.x_x64-setup.exe` из [последнего релиза](https://github.com/iamcoop1989-cell/rubypass/releases/latest)
2. Запустите установщик — при появлении UAC-запроса нажмите «Да» (приложению нужны права администратора для управления маршрутами)
3. Значок появится в системном трее

> **MSI или EXE?** Используйте EXE (NSIS) — он проще в установке. MSI предназначен для корпоративного развёртывания.

### Linux (x86_64)

1. Скачайте `RuBypass_x.x.x_amd64.AppImage` из [последнего релиза](https://github.com/iamcoop1989-cell/rubypass/releases/latest)
2. Сделайте файл исполняемым: `chmod +x RuBypass_*.AppImage`
3. Запустите: `./RuBypass_*.AppImage`
4. При первом использовании приложение попросит установить вспомогательный компонент (один запрос пароля)

## Использование

- **Включить/выключить** — клик по значку в трее или через меню «Включить / Выключить»
- **Обновить список адресов** — кнопка «Обновить список» в главном окне
- **Автозапуск** — переключатель в настройках, запускает приложение при старте системы
- **Выход** — если маршруты активны, приложение предложит очистить их перед закрытием

Обновления устанавливаются автоматически — при наличии новой версии приложение спросит разрешения и обновится само.

## Сборка из исходников

**Требования:** Rust 1.77+, Node.js не нужен.

```bash
git clone https://github.com/iamcoop1989-cell/rubypass
cd rubypass
cargo tauri build
```
